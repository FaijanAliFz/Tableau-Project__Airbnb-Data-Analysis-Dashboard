## 📊 Project Overview
This project presents an interactive Tableau dashboard analyzing Airbnb listing data. The dashboard provides insights into pricing trends, listing distribution, and market patterns, helping users understand the dynamics of the Airbnb rental market.

---

## 🎯 Objectives
- Analyze pricing trends across different listings  
- Understand distribution of properties and room types  
- Identify high-performing locations  
- Explore patterns in availability and demand  
- Support data-driven decision making  

---

## ⚙️ Dashboard Features
- Interactive filters for dynamic analysis  
- Price distribution across locations  
- Listing count by property type and room type  
- Geographic visualization of listings  
- Trend analysis for better market understanding  

---

## 🛠️ Tools & Technologies Used
- Tableau (Data Visualization)  
- Excel (Data Preparation)  

---

## 📈 Key Insights
- Pricing varies significantly based on location and property type  
- Certain neighborhoods show higher listing concentration  
- Entire homes/apartments dominate the market  
- Demand patterns vary across different regions
